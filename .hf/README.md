# Vektoryum

Raster gorselleri (logo, amblem, tabela, illustrasyon) olcum-gudumlu, kullanima
hazir **SVG / PDF / EPS / DXF / PNG** vektor ciktisina donusturen yapay zeka
motoru. Otomatik mod secimi, kenar temizleme (Schneider egri uydurma), renk
sadakati ve karsilastirmali onizleme icerir.

Bu Space `engine/Dockerfile` mantigini repo kokunden derler; kod GitHub reposundan
otomatik senkronlanir (`.github/workflows/hf-deploy.yml`).
