# F99-7 Print-Ready Validator

This phase defines a finite, fail-closed contract for validating production SVG, PDF, EPS and DXF artifacts before delivery.

## Qualified evidence

The contract requires at least 24 uniquely identified artifact sets. Every set contains one immutable digest for each required format and finite structural validation results.

## Acceptance thresholds

- Required formats are exactly `svg`, `pdf`, `eps` and `dxf`.
- Every required format must parse and pass structural validation.
- Open paths, self-intersections, invalid fill rules, missing color profiles and unsupported features must be zero.
- Page, viewBox and unit consistency scores must be finite and at least `0.990`.
- Every evidence record must bind to a valid immutable SHA-256 artifact digest.

Missing formats, duplicate cases, invalid digests, non-finite or out-of-range metrics, negative defect counts, corpus shrinkage or any print defect fails closed.
