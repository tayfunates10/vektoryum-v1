# F99-5 Multi-Scale Processing

This phase defines a finite, fail-closed acceptance contract for preserving geometry, color, typography and fine detail across supported raster and output scales.

## Qualified evidence

The contract requires at least 24 uniquely named cases. Each case must contain ordered evidence for the finite scale set `0.25x`, `0.5x`, `1x`, `2x` and `4x`, spanning logos, badges, small text, monoline art, multicolor artwork and low-resolution sources.

## Acceptance thresholds

- Mean cross-scale fidelity score: at least `0.990`.
- p05 cross-scale fidelity score: at least `0.980`.
- p95 normalized scale drift: at most `0.020`.
- Missing scale levels: `0`.
- Scale-order regressions: `0`.
- Topology regressions: `0`.
- Fine-detail loss regressions: `0`.

Missing identifiers or scale levels, duplicate cases, unexpected scale labels, non-finite or out-of-range metrics, negative defect counts, corpus shrinkage, or any threshold regression fails closed.
